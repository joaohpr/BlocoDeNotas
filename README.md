BlocoDeNotas

Projeto de um Bloco de Notas simples desenvolvido em Java, criado com o objetivo de praticar conceitos fundamentais da linguagem e organização de projetos.

Sobre o projeto

Este projeto simula um bloco de notas básico, permitindo a criação e manipulação de notas.  
Ele foi desenvolvido como parte do processo de aprendizado em Java e versionamento com Git e GitHub.

Funcionalidades

- Criar notas
- Editar notas existentes
- Excluir notas
- Listar todas as notas
- Identificação das notas por um código 
- Sistema de usuário com login

Cada usuário possui:
- Nome
- Email
- Senha com  dígitos

Tecnologias utilizadas

- Java
- Git
- GitHub

Como executar o projeto

1. Clone o repositório:
   git clone git@github.com:joaohpr/BlocoDeNotas.git

2. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code)

3. Compile e execute a classe principal do projeto

Estrutura do projeto

- Código organizado em classes
- Uso de objetos para simular um banco de dados
- Um conjunto de notas para cada usuário

Objetivo do projeto

- Praticar lógica de programação
- Trabalhar com classes e objetos
- Aplicar conceitos de encapsulamento
- Aprender o fluxo de versionamento com Git
